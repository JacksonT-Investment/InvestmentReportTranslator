## 🚀 部署准备

### 环境要求
- ✅ Node.js >= 16.0.0
- ✅ npm >= 8.0.0
- ✅ 支持的操作系统：Windows/Mac/Linux

### 配置步骤
1. **配置API Key**
   ```bash
   cp .env.example .env
   # 编辑 .env 文件
   KIMI_API_KEY=your_api_key
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动服务**
   ```bash
   ./start.sh
   ```
